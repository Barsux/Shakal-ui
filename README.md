# Требования
## > Python3.9
#
#
# Порядок установки
``` bash
$ git clone https://github.com/Barsux/Shakal-ui
$ cd Shakal-ui
$ python3 -m pip install -r requirements.txt
```

# Порядок запуска
``` bash
$ python3 -m shakal
```
